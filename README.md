# riken-workshop
*This repository contains the code for the RIKEN workshop on Characterization-based Error Mitigation for Quantum Computation, using the qedma_riken package (provided as a pre-built .whl file).

*Clone the repository and run the main Jupyter Notebook qesem_lab.ipynb to begin the workshop.

*The required Python packages are listed at the start of the notebook, along with the installation command:  pip install "qiskit>=2.0.0" "qiskit-ibm-runtime>=0.40.0" "qiskit-aer>=0.17.1" "networkx>=3.5" "matplotlib==3.10.0" "tqdm>=4.67.1" "scipy" "numpy" "python-dotenv==1.2.1" "pylatexenc>=2.10" 

*A reference solution is available in qesem_lab_solution.ipynb.
